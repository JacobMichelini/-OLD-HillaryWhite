Emacs


Running Stuff
C-c! open interpreter (e.g. if you're in a python script, a python interpreter will appear.)
C-c C-c run whole file
C-c| run selected lines
C-space select lines

Buffers
C-x C-f open file (with a buffer)
C-x C-s save file
C-x C-i insert file at cursor position
C-x b create/switch buffers
C-x C-b show buffer list
C-x k kill buffer
C-X C-x close down emacs

Movement
M-f forward one word
M-b backward one word
C-a beginning of line
C-e end of line
C-v one page up
M-v scroll one page down
M-< beginning of text
M-> end of text
C-f forward character
C-b backward one character
C-p last line
C-n next line


Editing
M-n repeat next command n times
C-d delete a character
M-d delete a word
M-Del delete word backwards
C-k kill line

C-Space begin selection
C-W delete marked region
M-W copy marked region
C-y paste the most recently deleted or copied line/region

Windows
C-x 2 split window horizontally
C-x o change to next window
C-x 0 delete window
C-x 1 close all windows except the one the cursor is in
C-x ^ enlarge window
C-x shrink-window
C-x 4 0 kill buffer and window
C-x 5 2 make new frame
C-x 5 o change to other frame
C-x 5 0 kill frame


Important
C-x u undo
C-g Quit the running command
M-x revert-buffer RETURN undo all changes since last save

Search & Replace
C-s search forwards
C-r search backwards
C-g return to where your search started (works if you are still in search mode)
M-% query replace
   Space, y = replace this occurrence
   Del, n = don't replace
   . only replace this, then exit
   , replace and pause
   ! replace all
   ^ back to previous match
   RETURN or q to quit replace
     Search/Replace with RegEx:
        ^ beginning of line
        $ end of line
        \< beginning of a word
        \> end of a word


Help
C-h c what command does this keystroke invoke
C-h k what command does this keystroke invoke and what does it do
C-h f what does this function do
C-h v what's this variable and what is its values
